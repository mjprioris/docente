# Currículum Docente · Laura Jiménez

Web estática lista para publicar en GitHub Pages.

## Estructura

```
index.html                              ← página web principal
cv/
└── Anexo_CV_Docente_Laura_Jimenez.docx ← CV descargable
publicaciones/
├── TC01.pdf    ← Comunicación escrita ICESD 2021
├── TC02.pdf    ← Comunicación oral ICESD 2021
└── TC03.pdf    ← Comunicación oral Nodos del Conocimiento 2021
```

## Pasos para publicar

1. Sube los 3 PDFs en `publicaciones/` con los nombres exactos: `TC01.pdf`, `TC02.pdf`, `TC03.pdf`
2. Sube todo el repositorio a GitHub
3. Ve a **Settings → Pages → Source: main / root**
4. En unos minutos la web estará en `https://tu-usuario.github.io/nombre-repositorio`
